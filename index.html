API V2 Documentation

API ini menggunakan endpoint V2 yang mengambil ID post dari URL Threads:

GET https://www.dolphinradar.com/api/threads/post_detail/{post_id}

Contoh penggunaan:

https://www.dolphinradar.com/api/threads/post_detail/DN7QaQkATv7?xmt=AQF0OtHSzHZnbK3tH8eoJydu9W_1G_aal4306kSzZjjmuw&slof=1

Response JSON akan berisi data postingan dan media.
