API Documentation

Anda juga dapat mengakses API ini langsung melalui endpoint berikut:

GET /api/threads?url={threads_url}

Contoh penggunaan dengan curl:

curl -X 'GET' \
'https://your-vercel-app.vercel.app/api/threads?url=https://www.threads.net/@yoda_erika/post/DN53X7Ik3al' \
-H 'accept: application/json'

Response JSON akan berisi:

{
  "success": true,
  "data": {
    "username": "otaku_anime_indonesia",
    "full_name": "Otaku Anime Indonesia",
    "caption": "Kerjanya nonton apa?!",
    "likes": 857,
    "media": [
      {
        "type": "video",
        "url": "https://video-url.mp4"
      }
    ]
  }
}
